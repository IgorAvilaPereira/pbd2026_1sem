# Lista

## Usuários

**1.** Crie uma stored procedure para **cadastrar um novo usuário**, recebendo nome, email e senha.

**2.** Crie uma função que **liste todos os usuários cadastrados**.

**3.** Crie uma função que **busque um usuário pelo email**.

**4.** Crie uma função que **retorne apenas o nome de um usuário a partir do id**.

**5.** Crie uma stored procedure para **alterar a senha de um usuário**.

---

## Equipamentos

**6.** Crie uma stored procedure para **cadastrar um novo equipamento**.

**7.** Crie uma função que **liste todos os equipamentos**.

**8.** Crie uma função que **busque um equipamento pelo id**.

**9.** Crie uma stored procedure para **atualizar a descrição de um equipamento**.

**10.** Crie uma stored procedure para **remover um equipamento**.

---

## Serviços

**11.** Crie uma stored procedure para **abrir um novo serviço**, informando título, descrição, criador e responsável.

**12.** Crie uma função que **liste todos os serviços cadastrados**.

**13.** Crie uma função que **liste apenas serviços ainda não finalizados**.

**14.** Crie uma função que **liste apenas serviços finalizados**.

**15.** Crie uma função que **busque um serviço pelo id**.

---

## Atualização de serviços

**17.** Crie uma stored procedure para **finalizar um serviço**.

**18.** Crie uma stored procedure para **alterar a descrição de um serviço**.

**19.** Crie uma stored procedure para **alterar o título de um serviço**.

**20.** Crie uma função que **conte quantos serviços existem no sistema**.

## Status

**23.** Crie uma função que **retorne o último status registrado de um serviço**.
